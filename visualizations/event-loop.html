Journal

Event Loop Node.js : Le moteur V8 execute le JavaScript sur un seul thread via la Call Stack. Les operations asynchrones (timers, I/O) sont deleguees a libuv. Les callbacks sont places dans la Macrotask Queue, tandis que les Promises et process.nextTick vont dans la Microtask Queue (prioritaire). Choisissez un scenario et observez l'ordre d'execution.
