Journal

NestJS Request Lifecycle : Chaque requete traverse : Middleware (comme Express) → Guard (autorisation) → Interceptor avant (logging, cache) → Pipe (validation, transformation) → Handler (logique metier) → Interceptor apres (transformation reponse) → Exception Filter (gestion d'erreurs, chemin alternatif).
